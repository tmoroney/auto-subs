# DaVinci Resolve Scripting API Reference

> **This file is intentionally a stub. The API reference lives in [`resolve-api.txt`](resolve-api.txt).**
>
> Blackmagic ships the reference as a plain-text, monospace column-aligned document. Rendering
> it as Markdown collapses the line breaks and breaks the column alignment (method names,
> return types, and descriptions), making it unreadable. So the reference is kept as `.txt`
> and treated as the source of truth.
>
> Open **`resolve-api.txt`** in a plain-text / monospace viewer. Re-sync it from your Resolve
> install after an update with `scripts/update-resolve-api.sh` (macOS/Linux) or
> `scripts/update-resolve-api.bat` (Windows).
>
> *(This `.md` can be deleted; it remains only as a pointer.)*
